# SimDesk UI 行为规范

> 此文档用于在无法运行 Windows 环境时，逐条对照验证 UI 行为

## 1. 系统托盘

| # | 行为 | 预期 |
|---|------|------|
| 1.1 | 程序启动 | 任务栏右侧出现 SimDesk 图标，无主窗口 |
| 1.2 | 右键托盘图标 | 弹出菜单：开机自启、默认选项、打开日志、关于、退出 |
| 1.3 | 开机自启勾选 | 注册表 HKCU\Run 写入 SimDesk 项 |
| 1.4 | 默认选项 → 拖入操作 | 子菜单显示"复制"/"移动"二选一 |
| 1.5 | 默认选项 → 盒子类型 | 子菜单显示"常规收纳盒"/"链接收纳盒"二选一 |
| 1.6 | 打开日志 | 弹出 LogViewerWindow |
| 1.7 | 关于 | 弹出 AboutWindow（版本 0.1.0） |
| 1.8 | 退出 | 保存所有盒状态后退出 |

## 2. 收纳盒悬浮窗

| # | 行为 | 预期 |
|---|------|------|
| 2.1 | 窗口外观 | 无边框、圆角(8px)、悬浮于桌面、不出现在任务栏 |
| 2.2 | 标题栏 | 左侧显示收纳盒名称，右侧⚙和▲按钮 |
| 2.3 | 拖动窗口 | 鼠标在标题栏按下拖动可移动窗口 |
| 2.4 | 双击名称 | 名称变为 TextBox，可编辑改名 |
| 2.5 | 改名 Enter | 提交新名称 |
| 2.6 | 改名 Escape | 取消改名，恢复原名称 |
| 2.7 | 改名失焦 | 自动提交新名称 |
| 2.8 | 位置锁定 | 锁定后窗口不可拖动 |

## 3. 设置弹出面板

| # | 行为 | 预期 |
|---|------|------|
| 3.1 | 点击⚙ | 在按钮下方弹出设置面板（不弹新窗口） |
| 3.2 | 点击面板外部 | 面板自动关闭 |
| 3.3 | 图标/列表切换 | RadioButton 切换显示模式 |
| 3.4 | 图标大小 | 输入框可设置 16-256px |
| 3.5 | 背景色选择 | 点击取色格弹出简化颜色面板 |
| 3.6 | 背景图选择 | 点击打开 Windows 文件选择器 |
| 3.7 | 背景图清除 | 清除背景图设置 |
| 3.8 | 透明度滑块 | 拖动改变窗口透明度，实时预览 |
| 3.9 | 位置锁定 | CheckBox 切换锁定状态 |
| 3.10 | 隐藏 | 窗口隐藏 |
| 3.11 | 解散 | 弹出确认对话框，确认后解散 |

## 4. 折叠/展开

| # | 行为 | 预期 |
|---|------|------|
| 4.1 | 点击▲ | 窗口高度动画收缩到标题栏 |
| 4.2 | 点击▼ | 窗口高度动画展开到原始高度 |
| 4.3 | 按钮图标 | 折叠时显示▼，展开时显示▲ |

## 5. 拖放文件

| # | 行为 | 预期 |
|---|------|------|
| 5.1 | 拖文件到常规盒 | 文件复制或移动到盒目录 |
| 5.2 | 拖文件到链接盒 | 在盒目录创建符号链接 |
| 5.3 | 拖目录到常规盒 | 递归复制目录 |
| 5.4 | 拖目录到链接盒 | 创建目录符号链接 |

## 6. 日志查看器

| # | 行为 | 预期 |
|---|------|------|
| 6.1 | 事件列表 | 按时间倒序显示，每页 50 条 |
| 6.2 | 事件节点 | 显示图标+时间+盒子名+摘要 |
| 6.3 | 解散节点展开 | 显示每个文件的还原位置 |
| 6.4 | 点击还原路径 | 在资源管理器中定位文件 |
| 6.5 | 筛选 | 可按事件类型/盒子筛选 |
| 6.6 | 分页 | 上一页/下一页按钮 |

## 7. 右键菜单（Shell Extension）

| # | 行为 | 预期 |
|---|------|------|
| 7.1 | 桌面空白处右键 | 显示"新建空收纳盒" |
| 7.2 | 文件右键 | 显示"新建收纳盒（含所选文件）" |
| 7.3 | 文件夹右键 | 显示"作为新收纳盒" |
| 7.4 | 点击菜单项 | 启动或通知 SimDesk 主实例执行操作 |
